# Adım Sayar

Bu proje tek sayfalık bir adım takip uygulamasıdır.

## GitHub Pages için yayın

1. Bu repo'yu GitHub'a yükleyin.
2. GitHub repository ayarlarında **Settings → Pages** bölümüne gidin.
3. **Source** olarak **GitHub Actions** seçin.
4. Ana branch'e push yaptıktan sonra site otomatik yayınlanır.

## Notlar
- Uygulama `localStorage` ile verileri saklar.
- Telefon için QR alanı, site URL'sini gösterir.
- GitHub Pages üzerinde `https://<kullanici>.github.io/<repo>/` şeklinde çalışır.
